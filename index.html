P1: CSE Simulator

Developed for EECS 440 - XR and Society at the University of Michigan.

An interactive VR experience where the user becomes a computer science student in a computer laboratory (with a spooky twist!)

Technologies Utilized:
- Unreal Engine 5: Used as the primary game engine to build the environment, handle physics, and implement real-time interactions.
- Blueprints (Unreal Engine 5 Visual Scripting): Utilized to create the logic behind interactions, including object affordances, event triggers, and NPC behaviors.
- JIRA: Used for project management, tracking tasks, and organizing milestones.
