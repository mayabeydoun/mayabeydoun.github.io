Mini 2D Game Engine

Developed for EECS 498 - Game Engine Architecture at the University of Michigan, this miniature game engine coded in C++ utilizes SDL, GLM, Lua with LuaBridge, and OpenCV. Custom features include an avatar creation tool and real-time face tracking, which enables players to place themselves in the game world.
